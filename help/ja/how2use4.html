簡単な導入方法 / 外部ファイル

外部ファイル

ねこープロジェクトII単体でもそれなりの動作をしますが、実機 BIOS 等のファイルを用意するとより実機に近づきます。
Xnp2 ではこれらのファイルを$(HOME)/.np2/ ディレクトリに置く事で、起動時に自動的に検出し使用します。

bios.rom

実機の BIOS ROM をファイル化した物です。
PC-9801 の MS-DOS 上で動作する専用のツールでファイル化して下さい。

ダウンロード: getbios.lzh (PC-9801 シリーズ MS-DOS 用)

FONT.ROM

実機のフォント ROM をファイル化した物です。
PC-9801 の MS-DOS 上で動作する専用のツールでファイル化して下さい。

ダウンロード: getbios.lzh (PC-9801 シリーズ MS-DOS 用)

FONT.ROM の他に BMP 形式のフォントファイルも使用できます。

sound.rom

サウンドボードに搭載される BIOS ROM をファイル化した物です。
PC-9801 の MS-DOS 上で動作する専用のツールでファイル化して下さい。

ダウンロード: getbios.lzh (PC-9801 シリーズ MS-DOS 用)

2608_bd.wav 〜 2608_rim.wav

OPNA に搭載されているリズム音源を wave 形式で録音したファイルです。
ねこープロジェクトで使用するには 44KHz 16bit モノラル 無圧縮形式である必要があります。

OPNA に搭載されているリズムは以下の通りです。

ファイル名対応する音色
2608_bd.wavバスドラム
2608_sd.wavスネアドラム
2608_top.wavシンバル
2608_hh.wavハイハット
2608_tom.wavタム
2608_rim.wavリムショット
