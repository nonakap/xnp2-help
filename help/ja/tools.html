ツール

make_hd

ぶるー氏作のPC-9801シリーズ上でフロッピーディスクをD88形式でファイル化するツールです。
PC-8801mkIISRエミュレータ P88SRに含まれます。

関連webpage: ぶるー牧場

DITT

cisc氏作のPC-ATマシン上でフロッピーディスクをD88形式でファイル化するツールです。

関連webpage: M88のページ

Virtual Floppy Image Converter

RED氏作のディスクイメージコンバートツールです。
ディスクイメージをコンバートするツールですが、実ディスクからのコンバートも可能です。

関連webpage: Vector - Virtual Floppy Image Converter

ディスク イマージュ

長谷川氏作のWindows95/98用のディスクイメージ化ツールです。
PC-9801シリーズで一般的に使用する1.2MBディスクをPC-ATでイメージ化する場合 3モードドライブが必要です。

関連webpage: Vector - ディスク イマージュ
